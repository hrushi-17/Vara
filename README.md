## ✨ Highlight Features

### 🛍️ Premium User Interface
A sleek, high-quality, and dynamic design utilizing a meticulously curated color palette (olive, tan, cream, sand) to evoke a sense of luxury and timeless craftsmanship. 

### 📱 Flawless Responsiveness
Built with a mobile-first approach, featuring seamless CSS breakpoints. Whether accessed on an iPhone, an iPad, or a high-resolution ultra-wide desktop monitor, the layout remains completely flawless.

### 🔐 Robust Authentication Flow
Fully functional multi-user authentication system equipped with duplicate account prevention mechanisms, elegant modal interfaces, and robust error handling.

### 👤 User Dashboard & Account Management
An interactive "My Account" dashboard allowing authenticated users to manage editable profile fields securely and efficiently.

### 🛒 Seamless Cart Experience
A sophisticated side-drawer cart design delivering real-time updates and an integrated checkout flow utilizing WhatsApp for effortless order completion.

## 🚀 Quick Start Guide

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/hrushi-17/Vara.git
   ```

2. **Navigate to the directory**
   ```bash
   cd vara
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
   *Your application will typically be running on http://localhost:5173.*
