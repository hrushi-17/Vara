<h1 align="center">👜 Vara - Premium Leather Goods E-Commerce</h1>

<div align="center">
  <strong>A high-performance, modular React application providing a premium, Flipkart-inspired user experience for luxury leather goods.</strong>
</div>

<br />

## ✨ Features

- **🛍️ Premium User Interface:** A sleek, high-quality design inspired by leading e-commerce platforms like Flipkart.
- **🔐 Robust Authentication:** Fully functional multi-user authentication system with duplicate account prevention mechanisms.
- **👤 User Dashboard:** Interactive "My Account" dashboard allowing users to manage editable profile fields securely and efficiently.
- **📜 Order Tracking:** Intelligent mock order history system that rigorously tracks WhatsApp-integrated checkouts.
- **⚙️ Modular Architecture:** Built with React 19 & Vite for lightning-fast performance, hot module replacement, and component reusability.
- **📱 Responsive Design:** Flawlessly responsive layout tailored for pristine appearance across mobile, tablet, and desktop screens.

## 🚀 Quick Start

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hrushi-17/Vara.git
   ```

2. **Navigate into the project directory**
   ```bash
   cd vara
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
   *Your application will typically be running on http://localhost:5173.*
